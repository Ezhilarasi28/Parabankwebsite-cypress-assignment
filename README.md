**Setup-cypress**

**Prerequisties:**

- Ensure Node.js is installed.

**setting**:
Install Cypress
npm install cypress --save-dev

**Run the Project**
1. **clone the project:**
   git clone <URL>
2. **Project directory:**
   cd CypressAssignment
3. **Cypress open:**
   npx cypress open

4. **Run test in Cypress** --  After Cypress opens, click on the test file to run it.
